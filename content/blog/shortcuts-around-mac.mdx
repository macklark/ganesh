I've always found that keyboard shortcuts make navigation faster than reaching for the mouse and clicking through everything manually. So these are some of the shortcuts I've put together for myself to move between apps and tabs. They might not be perfect, but they're a start for my first blog post.

Using [Raycast](https://www.raycast.com/) instead of Spotlight makes navigation even more powerful, since it provides a lot more functionality than Spotlight.

## Switching Between Apps

- `⌘ + ⌥ + C` — switch to Chrome
- `⌘ + ⌥ + Z` — switch to Zed
- `⌘ + ⌥ + T` — switch to Terminal
- `⌘ + ⌥ + L` — switch to Claude

## Navigating Around Zed

- `⌘ + N` — create new file
- `⌘ + S` — save file
- `⌘ + W` — close a file
- `⌘ + J` — toggle terminal to appear
- `⌘ + B` — toggle file explorer
- `⌘ + shift + P` — toggle command execution
- `⌘ + P` — toggle file search explorer

## Navigating Around Chrome

- `⌘ + T` — new tab
- `⌘ + W` — close tab
- `⌘ + L` — focus search
- `⌘ + back` — navigate back to previous page
- `⌘ + forward` — navigate forward to next page


Try them out — trust me, they make navigation faster, and over time they'll become muscle memory.
